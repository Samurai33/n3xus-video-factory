# Create Shorts Batch

Use esta skill para gerar lotes de Shorts/Reels/TikTok.

## Entrada esperada

- produto;
- público;
- promessa;
- dor;
- tom;
- quantidade de vídeos.

## Padrão de saída

Para cada vídeo:

```md
# Vídeo 01

## Pilar
Dor / Demonstração / Autoridade / Prova / Comparação

## Hook
Frase de 0 a 2 segundos.

## Roteiro
Texto curto de 18 a 32 segundos.

## Direção visual
O que aparece na tela.

## Legendas
Frases curtas para tela.

## CTA
Ação única.

## Variação A/B
Outro hook ou outro CTA.
```

## Regras

- Uma ideia por vídeo.
- Hook forte nos primeiros 2 segundos.
- Não usar promessa impossível.
- Priorizar prova visual.
- Para HotLead, usar linguagem SaaS premium, direta e vendedora.
