# Hooks

Biblioteca de hooks para Shorts/Reels/TikTok.

## Padrão

- 0 a 2 segundos.
- Dor clara.
- Curiosidade forte.
- Sem promessa impossível.

## Exemplos HotLead

- Seu concorrente está chegando antes de você.
- Você não precisa de mais leads. Precisa de leads melhores.
- O problema não é tráfego. É intenção invisível.
